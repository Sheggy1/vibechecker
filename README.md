⚡ VibeCheck — Sentiment Engine for Social Media Managers

Paste any customer comment and instantly get an AI-powered 
vibe analysis before you reply.

🔍 Main Features
- Sentiment detection — POSITIVE / NEGATIVE / NEUTRAL / MIXED
- Confidence score — 0–100% with color-coded progress bar
- Energy level — HIGH / MEDIUM / LOW
- Vibe summary — one-line AI insight tailored for social context
- Session history — last 5 checks stored during the session
- Test suite — 9 integration tests with mocked API responses

🛠 Tech Stack
- Frontend: React 18 + Vite
- Backend: Node.js + Express (local API proxy)
- AI: Anthropic Claude API (claude-sonnet)
- Testing: Vitest + jsdom
- Styling: Inline styles with CSS variables

🚀 Quick Start
npm install
npm run dev
→ http://localhost:5173
